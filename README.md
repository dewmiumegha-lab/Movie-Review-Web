# 🎬 Movie Review App (Firebase Integrated)

A modern, sleek, and fully functional **Movie Review Web Application** built with a focus on high-end UI/UX and real-time data management. This app allows users to add, edit, delete, and sort movie reviews with a premium cinema-themed interface.

---

## ✨ Features

- **Real-time Database:** Powered by **Google Firebase Firestore** for instant data synchronization.
- **Full CRUD Operations:**
  - **Create:** Add new movies with ratings, directors, and release dates.
  - **Read:** View a dynamically updated list of movie reviews.
  - **Update:** Edit existing movie names and ratings.
  - **Delete:** Remove reviews from the database.
- **Advanced UI/UX:**
  - Cinema-inspired **Dark Theme** (Black, Red, and Gold).
  - **Glassmorphism** effects and interactive hover animations.
  - Fully responsive design for mobile and desktop views.
- **Dynamic Sorting:** Sort your collection by Name, Rating, Director, or Date.

---

## 🚀 Tech Stack

*   **Frontend:** HTML5, CSS3 (Advanced Grid/Flexbox, Animations)
*   **Scripting:** JavaScript (ES6 Modules, Async/Await)
*   **Backend as a Service (BaaS):** Firebase Firestore

1. **Clone the repository:**
   ```bash

Firebase Configuration:

Go to the Firebase Console.

Create a new project and set up Firestore Database.

Copy your Firebase config object and replace it in the script.js file:

JavaScript
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  projectId: "YOUR_PROJECT_ID",
  // ... rest of your config
};
Run the App:

Simply open index.html in your browser or use a Live Server extension in VS Code.
